Agenda Telefónica usando c#

Este proyecto es una pequeña agenda telefónica en consola, donde puedes guardar, buscar, listar y borrar contactos de forma sencilla. 
El programa muestra un menú con opciones. Solo escribes el número de la acción que quieres hacer y sigues las instrucciones en pantalla.
Los contactos se guardan en una lista interna usando nombre y teléfono.

¿Qué hace el programa?

-Agregar un contacto con nombre y número.
-Buscar un contacto por su nombre.
-Listar todos los contactos guardados.
-Eliminar un contacto.
-Salir del programa cuando termines.
